General Concepts
================
The |product| generally revolves around three major concepts, each having their own SDK
client.

* **User**: the user authenticated to interact with the |company| data. The user has
    access to all the resources and capabilities that are also available on the
    |platform|.
* **Dataset**: the space where the data it self lives. Datasets can be reused across
    multiple projects and contains no labels or annotations them selves.
* **Project**: the space where labels, annotations, and reviews lives. Projects can be
    linked to multiple datasets.
